# Description
## 實務專題競賽網站
在原有的實務專題檢索系統中加入一個投票系統
讓系上教授與學生可以在一定期間內對自己喜歡的專題題目投票

![](http://i.imgur.com/RFvZkEz.jpg)
![](http://i.imgur.com/lfrUy95.png)

1. 原專題檢索系統的新增、查詢、修改

2. 投票系統

- 系上教授、學生投票
- 管理端可以查詢成績
- 教授:學生 = 8:2
- 最多六票

3. 後端

- 管理專題生的專題資料
- 把每次投票定義成活動
  - **年度、時程**
